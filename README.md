## 使用的技术

- Springboot
- SpringSecurity
- EasyExcel
- Vue
- Bootstrap
- Vue Quill编辑器



## 特性

- 样式简洁，支持自定义网站名称/Logo等信息
- 支持第三方登录认证
- 支持多用户(https://gooodh/nav/用户名)
- 支持多级分类（导航菜单）
- 支持Excel批量导入应用（导航项）
- 输入应用网址，自动爬取Logo/标题/描述信息
- ...

参考VanNav部分样式和功能。
